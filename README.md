# BCR Tree Pipeline

A pipeline for generating B-cell receptor (BCR) repertoire evolution trees using MIXCR and GCTree.

## Description
This pipeline takes `FASTQ` files (bulk BCR sequencing data) as input and generate BCR evolutional trees. 

## Prerequisites

- MIXCR: See [installation guide](https://mixcr.com/mixcr/getting-started/installation/)
- GCTree: See [github GCTree](https://github.com/matsengrp/gctree), paper [Using Genotype Abundance to Improve Phylogenetic Inference](https://academic.oup.com/mbe/article/35/5/1253/4893244) and official [documentation](https://matsen.group/gctree/)

## Installation

```bash
git clone <repository-url>
cd bcrtree_pipeline
```
